## Installation

* change
```sh
.env.example
```
to
```sh
.env
```
### contents of the .env file
* GET TOKEN [DEVELOVER PORTAL](https://discord.com/developers)
```sh
TOKEN=
```
* OPTIONAL
```sh
GUILD_ID=
```
* GET CLIENT ID [DEVELOVER PORTAL](https://discord.com/developers)
```sh
CLIENT_ID=
```
* GET API [Google Ai Studio](https://aistudio.google.com/app/apikey)
```sh
GEMINI_API=
```
## INSTALL & RUN
* npm
```sh
npm i
```
or
```sh
npm install
```

* starting
```sh
node .
```
or
```sh
node index.js
```
